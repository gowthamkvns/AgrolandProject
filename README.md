# AgrolandProject

## Project Overview
AgrolandProject is a decentralized application (DApp) built on the **Algorand Blockchain**. It aims to bring transparency and trust to agricultural supply chains by recording crop data, ownership transfers, and quality certifications on-chain.

## Objective
The goal is to leverage Algorand's high-speed, low-cost transactions to:
- Track agricultural produce from farm to consumer.
- Issue NFT-based certificates for organic or fair-trade verification.
- Automate payments between farmers and buyers using Smart Contracts.

## Project Structure
- **/contracts**: Contains PyTeal / Beaker smart contract logic for escrow and asset management.
- **/scripts**: Python/JS scripts for deploying contracts and interacting with the Algorand Testnet.
- **/frontend**: A React-based dashboard for farmers and distributors to manage inventory.

## Tech Stack
- **Blockchain:** Algorand (Layer 1)
- **Smart Contracts:** Python (PyTeal / Beaker)
- **SDK:** Algorand Python SDK
- **Environment:** AlgoKit / Sandbox
